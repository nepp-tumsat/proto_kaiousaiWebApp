# 開発ガイド

## ローカル開発サーバーの起動

### 基本的な起動方法

```bash
npm run dev
```

ブラウザで `http://localhost:3000/` にアクセスします（Next.js のデフォルトポート）。

### スマホからアクセスする方法

1. **PCとスマホを同じWi-Fiネットワークに接続**

2. **開発サーバーを LAN から受け付けるように起動**
   ```bash
   npx next dev -H 0.0.0.0
   ```
   または `package.json` に同様のスクリプトを追加して利用してもよい。

3. **PCのローカルIPアドレスを確認**
   - macOS: `ifconfig | grep "inet " | grep -v 127.0.0.1`
   - Windows: `ipconfig` で「IPv4アドレス」を確認
   - Linux: `ip addr show` または `hostname -I`

4. **スマホのブラウザでアクセス**
   ```
   http://[PCのIPアドレス]:3000
   ```
   例: `http://192.168.1.100:3000`

### 注意事項

- PCとスマホが同じWi-Fiネットワークに接続されている必要があります
- ファイアウォールでポート3000がブロックされている場合は、許可設定が必要です
- 開発サーバーは起動中である必要があります

## ビルドとプレビュー

### 本番用ビルド

```bash
npm run build
```

静的エクスポートの設定により、成果物は `out/` に出力されます。

### 本番サーバーでのプレビュー（Next start）

```bash
npm run preview
```

`next start` を実行します（静的エクスポートのみの運用では、ローカルは `out/` を静的サーバーで開く方法もあります）。

## トラブルシューティング

### スマホからアクセスできない場合

1. **PCのIPアドレスを再確認**
2. **`-H 0.0.0.0` で起動しているか確認**
3. **ファイアウォール設定を確認**
   - macOS: システム環境設定 → セキュリティとプライバシー → ファイアウォール
4. **同じWi-Fiネットワークに接続されているか確認**
5. **開発サーバーが起動しているか確認**

### ポートが使用中の場合

別のポートで起動します。

```bash
npx next dev -H 0.0.0.0 -p 3001
```
