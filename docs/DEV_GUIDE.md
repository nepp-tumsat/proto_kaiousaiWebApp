# 開発ガイド

## ローカル開発サーバーの起動

### 基本的な起動方法

```bash
npm run dev
```

ブラウザで `http://localhost:5173/` にアクセスします。

### スマホからアクセスする方法

1. **PCとスマホを同じWi-Fiネットワークに接続**

2. **開発サーバーを起動**
   ```bash
   npm run dev
   ```

3. **PCのローカルIPアドレスを確認**
   - macOS: `ifconfig | grep "inet " | grep -v 127.0.0.1`
   - Windows: `ipconfig` で「IPv4アドレス」を確認
   - Linux: `ip addr show` または `hostname -I`

4. **スマホのブラウザでアクセス**
   ```
   http://[PCのIPアドレス]:5173
   ```
   例: `http://192.168.1.100:5173`

### 注意事項

- PCとスマホが同じWi-Fiネットワークに接続されている必要があります
- ファイアウォールでポート5173がブロックされている場合は、許可設定が必要です
- 開発サーバーは起動中である必要があります

## ビルドとプレビュー

### 本番用ビルド

```bash
npm run build
```

`dist/`ディレクトリにビルド結果が出力されます。

### ビルド結果のプレビュー

```bash
npm run preview
```

ビルド後のファイルをローカルで確認できます。

## トラブルシューティング

### スマホからアクセスできない場合

1. **PCのIPアドレスを再確認**
2. **ファイアウォール設定を確認**
   - macOS: システム環境設定 → セキュリティとプライバシー → ファイアウォール
3. **同じWi-Fiネットワークに接続されているか確認**
4. **開発サーバーが起動しているか確認**

### ポートが使用中の場合

`vite.config.ts`の`server.port`を変更してください。

```typescript
server: {
  host: true,
  port: 3000  // 別のポート番号
}
```
