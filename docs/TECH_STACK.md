# 使用技術スタック

## 概要

海王祭Webアプリは、モダンなWeb技術を使用して構築されたPWA（Progressive Web App）です。

## フロントエンド

### コア技術

- **React 18.2.0**
  - UI構築のためのライブラリ
  - コンポーネントベースのアーキテクチャ

- **TypeScript 5.2.2**
  - 型安全性を提供
  - 開発時のエラー検出を強化

- **Vite 5.0.8**
  - 高速なビルドツール
  - HMR（Hot Module Replacement）対応
  - 開発サーバー

### UIライブラリ

- **Leaflet 1.9.4**
  - インタラクティブマップの表示
  - モバイル対応のタッチ操作

- **react-leaflet 4.2.1**
  - React用のLeafletラッパー
  - Reactコンポーネントとしてマップを実装

### スタイリング

- **CSS Modules**
  - コンポーネント単位のスタイル管理
  - スコープ化されたCSS

- **カスタムCSS変数**
  - マリンブルー (#003366) を基調としたテーマカラー

## PWA機能

- **vite-plugin-pwa 0.17.4**
  - Service Workerの自動生成
  - オフライン対応
  - アプリインストール機能

## 開発ツール

### ビルド・バンドル

- **Vite**
  - ESBuildによる高速ビルド
  - コード分割
  - アセット最適化

### リンター・フォーマッター

- **ESLint 8.55.0**
  - コード品質チェック
  - TypeScript対応

- **TypeScript ESLint**
  - TypeScript専用のリンタールール

## デプロイ

- **GitHub Pages**
  - 静的サイトホスティング
  - HTTPS対応

- **GitHub Actions**
  - CI/CDパイプライン
  - 自動ビルド・デプロイ

## データ管理

- **JSONファイル**
  - ダミーデータの管理
  - フロントエンド内でのデータ保持

## パッケージ管理

- **npm**
  - 依存関係の管理
  - パッケージのインストール

## 技術選定の理由

### React + TypeScript
- 型安全性と開発効率のバランス
- モダンな開発体験

### Vite
- 高速な開発サーバー
- シンプルな設定

### Leaflet.js
- 軽量でモバイル対応
- カスタムマップ画像の表示が容易

### PWA
- アプリストア不要でインストール可能
- オフライン対応

## ブラウザ対応

- **モバイルブラウザ（優先）**
  - iOS Safari
  - Android Chrome
  - その他のモダンブラウザ

- **PWA機能**
  - Service Worker対応ブラウザ
  - ホーム画面追加機能

## パフォーマンス

- **ビルドサイズ**
  - メインバンドル: ~300KB (gzip: ~92KB)
  - CSS: ~18KB (gzip: ~7KB)

- **最適化**
  - コード分割
  - 画像の遅延読み込み
  - Service Workerによるキャッシュ
