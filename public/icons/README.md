# PWAアイコンファイル

このディレクトリに以下のアイコンファイルを配置してください：

- `icon-192x192.png` - 192x192ピクセルのアイコン
- `icon-512x512.png` - 512x512ピクセルのアイコン

## アイコンの要件

- **形式**: PNG
- **サイズ**: 192x192 と 512x512
- **背景**: 透明またはマリンブルー (#003366)
- **デザイン**: 海王祭のロゴまたは関連するアイコン

## アイコン生成ツール

以下のオンラインツールを使用してアイコンを生成できます：
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

デモ用のプレースホルダーアイコンでも構いません。
